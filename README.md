# 🧿 决策助手 (Decision App)

**基于个人偏好校准的 AI 决策辅助工具。**

先通过 80 条跨领域理论评分，形成你独一无二的"决策指纹"。之后遇到任何犹豫不决的事——要不要追、该不该买、吃什么、穿什么——输入描述，AI 会用你的方式替你想清楚。

## 怎么用

1. **测评** → 给 80 条理论打分（0-5），形成你的决策指纹
2. **信念** → 添加你的核心原则（比如"我要减熵"），贯穿所有分析
3. **决策** → 选类别、描述困境，点"帮我分析"
4. **结果** → 个性化分析 + 关联理论 + 反面视角 + 行动建议 + 相似案例

## 理论覆盖

- 财富与风险（分散投资、能力圈、复利…）
- 心智与认知（沉没成本、锚定效应、幸存者偏差…）
- 处世之道（三思后行 vs 机不可失、知足常乐 vs 取法乎上…）
- 决策科学（汉隆剃刀、二阶思维、事前验尸…）
- 现代思潮（YOLO、断舍离、熵增定律、杠铃策略…）

每条理论都有对立观点供用户自由选择，不站队、不爹味。

## 技术

- React + TypeScript + Vite
- Tailwind CSS
- DeepSeek / Claude 双 AI 引擎
- 纯前端，数据存本地浏览器，不上传任何信息

## 跑起来

```bash
npm install
npm run dev
```

打开 http://localhost:5173 ，去设置页填入 API Key 即可使用。
